# This is data for my website

It is automatically generated by [Jekyll](https://github.com/jekyll/jekyll) into a static site. Blog posts are written using [JekyllStudio](https://github.com/tatey/JekyllStudio). You can visit the live site at [tatey.com](http://tatey.com).

## Installation

Install [Devbox](https://github.com/jetify-com/devbox) (Optional).

Clone the repository and bootstrap:

    $ git clone git://github.com/tatey/tatey.com.git
    $ cd tatey.com
    $ devbox shell # (if installed)
    $ _script/bootstrap

Serve:

    $ _script/serve

## Deploy

Install awscli and configure environment variables. Then deploy:

    $ _script/deploy
